# SIUX Studio

This is a React Component implementation of [SIUX Studio](https://siux.studio)

## Getting started

##### Node version

```bash
v16.16.0
```

##### Install

```bash
npm i
```

##### Run the development server

```bash
npm run storybook
```

##### Preview your application in browser

```bash
http://localhost:6006
```
